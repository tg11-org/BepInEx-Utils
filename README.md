# TG11 Utils

Shared utility plugin for Crime Simulator mods.

This package provides common helpers, logging utilities, and shared infrastructure
used by other TG11 mods.

## Features
- Centralized logging
- Shared config helpers
- Common utilities for Harmony patches

## Usage
This mod is intended as a dependency for other mods.
It does not add gameplay features on its own.

## Requirements
- BepInEx 5 (Mono)
